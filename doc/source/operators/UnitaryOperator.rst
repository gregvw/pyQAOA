UnitaryOperator
=================

Unitary operators have the property that :math:`U^\ast = U^{-1}` and :math:`\|U\|=1`, 
which implies that the operator has eigenvalues with unit magnitude. 
:math:`\|E\|=1,\;\forall\;E\in \sigma(U)`.
